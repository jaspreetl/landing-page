# Personal Landing Page

This repository contains a single-page personal site that uses a hero video background, a CV/projects section with a terracotta accent band, an about section with headshot, and a contact block. 

## Quick start

Local server:
```bash
python3 -m http.server 3000
# then visit http://localhost:3000
```

## Structure

- `index.html` – Inline React (via CDN + Babel) and all layout/styling. Sections:
  - Hero with video background and CTA.
  - CV (Work/Education).
  - Projects band (terracotta background).
  - About Me with bio and headshot.
  - Contact / Get in touch block and footer.
- `images/` – Media assets (hero video, headshot, favicon/logo).
